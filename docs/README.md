# Documentation

Sui Clean is an unofficial cleaned-up Sui repository. These docs explain the repository layout, build workflow, GitHub contribution process, and how this cleaned workspace relates to the official Sui project.

Official upstream project: <https://github.com/MystenLabs/sui>

## Start here

- `embedded_sources.md` — where embedded Sui and Move sources live
- `root_layout.md` — root folder structure

- `crates_domains.md` — source-domain rules
- `domain_commands.md` — `cargo xtask` commands
- `windows_build.md` — Windows build notes
- `github.md` — GitHub setup and PR expectations
- `github_release_checklist.md` — baseline checklist before tagging releases
- `license_attribution.md` — upstream licensing and attribution
