# Documentation

Sui Clean is an unofficial cleaned-up Sui repository. These docs explain the repository layout, build workflow, GitHub contribution process, and how this cleaned workspace relates to the official Sui project.

Official upstream project: <https://github.com/MystenLabs/sui>
